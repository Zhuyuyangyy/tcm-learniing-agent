---

## 🌟 项目亮点

### 核心技术
- **CrewAI 多智能体**：5个专业 Agent 协同工作（档案构建师 / 中医检索师 / 资源生成师 / 批判分析师 / 路径规划师），模拟中医专家团队辨证论治思维
- **RAG 知识增强检索**：结合向量数据库与知识图谱，确保回答的专业性和可溯源性
- **知识图谱**：构建中医脏腑经络关系网络，支持多跳推理与关联挖掘
- **用药安全审核**：内置十八反十九畏数据库，自动检测配伍禁忌

### 创新性
- 首次将 CrewAI 多智能体框架引入中医辨证论治领域
- 融合九种体质分类理论与 AI 精准个性化推荐
- 知识图谱 + RAG 双驱动，提升中医知识推理的可解释性

## 🚀 快速启动

### 前端
```bash
cd frontend
npm install
npm run dev
# 访问 http://localhost:5173
```

### 后端
```bash
# 使用项目虚拟环境
.\.venv\Scripts\python.exe -m uvicorn app.main:app --host 0.0.0.0 --port 8000
```

### 所需环境
- Python 3.10+
- Node.js 18+
- 讯飞星火 API Key 或 Dify API（用于 LLM 调用）

## 🧠 辨证论治流程

```
用户输入症状
    ↓
[档案构建师] 分析患者画像、体质倾向
    ↓
[中医检索师] + [资源生成师] 并行工作
    ↓
[批判分析师] 审核：十八反十九畏、逻辑一致性
    ↓
[路径规划师] 综合决策，输出最优治疗路径
    ↓
个性化处方 + 剂量建议 + 调养方案
```
