## 项目核心技术底子

- **5个专业Agent协作**：档案构建师 / 中医检索师 / 资源生成师 / 批判分析师 / 路径规划师
- **RAG知识检索**：ChromaDB向量数据库 + 知识图谱双驱动
- **知识图谱**：43节点，覆盖脏腑经络体质核心中医实体
- **用药安全**：37组十八反十九畏配伍禁忌自动审核
- **体质分类**：9种国标体质，45题问卷 + 雷达图报告
- **技术栈**：Vue3 + FastAPI + CrewAI + ECharts + Docker

## 使用建议

### PPT制作顺序
1. 先填05-团队介绍模板.md（最简单）
2. 参考01-商业计划书.md填充02-路演演讲稿.md的内容
3. 根据04-PPT制作规范.md制作PPT
4. 用06-PPT图片生成提示词.md生成配图
5. 用03-评委Q&A详细答案库.md准备答辩

### 答辩准备
1. 路演稿演练3遍以上，控制5分钟内
2. 熟悉评委Q&A每个问题的答案
3. 准备一个DEMO演示（体质测评或问诊）
4. 团队分工明确，答辩时配合默契
