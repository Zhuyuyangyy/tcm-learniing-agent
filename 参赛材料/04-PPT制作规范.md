# PPT制作规范
## 中医智能诊疗系统 TCM Mind-RAG | 互联网+大赛省级赛

> 版本：v1.0 | 2026年4月
> 用途：指导PPT视觉设计和制作

---

## 一、整体风格

### 配色方案

| 用途 | 颜色 | 色值 | 说明 |
|------|------|------|------|
| 主背景 | 深蓝黑 | `#0f1d35` | 科技感、沉稳 |
| 次背景 | 深靛蓝 | `#1a2744` | 卡片、模块背景 |
| 主强调色 | 青色 | `#00d4aa` | 标题、图标、关键数据 |
| 辅助强调 | 金色 | `#ffd700` | 亮点、成就数据 |
| 文字主色 | 白色 | `#ffffff` | 正文、标题 |
| 文字次色 | 浅灰蓝 | `#94a3b8` | 副标题、说明文字 |
| 警示色 | 红色 | `#ef4444` | 十八反十九畏警示 |
| 国风辅助 | 墨绿 | `#065f46` | 国风模式点缀 |

### 字体建议

| 用途 | 字体 | 大小 | 说明 |
|------|------|------|------|
| 标题 | 思源黑体 / Microsoft YaHei Bold | 36-44pt | 主标题 |
| 副标题 | 思源黑体 / Microsoft YaHei | 24-28pt | 页面标题 |
| 正文 | 思源黑体 / Microsoft YaHei | 16-18pt | 内容文字 |
| 数据大字 | DIN Alternate / Arial Bold | 48-72pt | 关键数字展示 |
| 英文 | Inter / Roboto | 同正文 | 英文内容 |

### 排版规范

- **页边距**：左右各60px，上下各40px
- **标题位置**：左上角或居中，高度不超过页面10%
- **内容区**：左右留白，文字不要太满
- **图片位置**：右侧或居中，保持视觉平衡
- **数据展示**：数字要大，颜色要醒目

## 三、制作工具推荐

### 推荐制作顺序

**第一步：Gamma.app（快速出初稿）**
- 访问 gamma.app
- 粘贴准备好的8页文字内容
- AI自动生成专业PPT初稿
- 耗时：5-10分钟出初稿

**第二步：PowerPoint精修**
- 下载PPTX文件到本地
- 调整数字大小和颜色
- 美化图表和布局
- 添加页码和logo

**第三步：关键图片制作**
- 架构图：draw.io（免费）
- 流程图：Canva
- UI图：截图+D标注
- Logo：Canva或PS

### Gamma.app使用技巧

1. **导入方式**：直接把8页文字粘贴进Gamma
2. **风格选择**：选择"科技感/深色"主题
3. **图片**：上传前可以用配图提示词生成
4. **导出**：下载PPTX后在PowerPoint中精修

## 五、注意事项

1. **不要照搬代码**：评委不一定懂代码，技术架构图要简化
2. **数据要醒目**：关键数字用大号字体+强调色
3. **保持视觉一致**：8页用同一套配色和字体
4. **预留Q&A时间**：PPT控制在4分钟内，留1分钟给评委翻页
5. **备份！**：比赛前把PPT存到U盘和云端，多重备份
